# Data Communications — 3. 물리층 개요 (1)

## 3.1 데이터와 신호

| 구분 | 데이터 | 신호 |
|---|---|---|
| 아날로그 | 음성, 온도, 시간 등 연속적인 값 | 연속적인 파형 |
| 디지털 | 0과 1로 저장된 데이터 | 이산적, 제한된 값 (0 또는 1) |

**변환 장치**
- **모뎀**: 디지털 데이터 ↔ 아날로그 신호
- **코덱**: 아날로그 데이터 ↔ 디지털 신호

## 3.2 주기 아날로그 신호

### 정현파 (Sine Wave)

아날로그 주기 신호의 가장 기본적인 형태

$$s = A \sin(2\pi ft + \phi)$$

| 요소 | 기호 | 설명 |
|---|---|---|
| 순간 진폭 | $s$ | 특정 시점의 신호 크기 |
| 최대 진폭 | $A$ | 신호의 최대 세기 |
| 주파수 | $f$ | 1초당 반복 횟수 (Hz) |
| 위상 | $\phi$ | 파형의 시작 위치 |

### 파장 (Wavelength)

단순 신호가 **한 주기 동안 진행하는 거리**

$$\lambda = \frac{c}{f}$$

- $c$: 전파 속도 (진공에서 빛의 속도 $3 \times 10^8$ m/s)
- $f$: 주파수

> 예) 빨간색 빛 ($f = 4 \times 10^{14}$ Hz) → $\lambda = \frac{3 \times 10^8}{4 \times 10^{14}} = 7.5 \times 10^{-7}$ m

### 복합 신호

실제 데이터 통신에 사용되는 신호는 복합 신호

- **주기적 복합 신호**: 이산적인 주파수 성분들로 분해 가능
- **비주기적 복합 신호**: 연속적인 주파수 범위를 가짐

### 대역폭 (Bandwidth)

$$\text{대역폭} = f_{\max} - f_{\min}$$

복합 신호에 포함된 **최고 주파수 - 최저 주파수**

## 3.3 디지털 신호

- 디지털 신호 = 무한대의 주파수를 갖는 복합 아날로그 신호
- 대역폭 = 무한대

### 전송 방식

| 방식 | 설명 | 특징 |
|---|---|---|
| 기저대역 (Baseband) | 디지털 신호를 그대로 채널로 전송 | 저대역 통과 채널 사용, LAN에 주로 사용 |
| 광대역 (Wideband) | 디지털 신호를 아날로그로 변조 후 전송 | 밴드패스 채널 사용, 변복조 필요 |

**기저대역 전송**
- 저대역 통과 채널(low-pass channel) 사용 — 주파수 0부터 시작
- 아날로그 변환 없이 그대로 전송
- 동축케이블, 광섬유, LAN에서 사용

**광대역 전송**
- 디지털 신호 → 아날로그 신호로 변조(modulation)
- 밴드패스(bandpass) 채널 사용
